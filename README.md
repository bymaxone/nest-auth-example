# nest-auth-example

Reference application for [`@bymax-one/nest-auth`](https://github.com/bymax-one/nest-auth) — a full-stack authentication library for NestJS + Next.js. This repo demonstrates every feature the library ships: JWT refresh rotation, MFA (TOTP), OAuth (Google), invitations, brute-force protection, WebSocket auth, multi-tenancy, platform admin, and more.
